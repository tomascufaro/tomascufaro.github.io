Vibration inherently exists within human life. In each process of our environment vibration are involved. Wave phenomena is pretty related to vibration. Understanding wave phenomena trough vibration and how can we use this info as descriptor are the main goals of this work

Mechanics vibration measurement, study purpose of this project, might be a important tool for industry development. For instance each machine that has an engine present a rotatory system that generates vibration. Those vibration can be usually described as va estar atravesado por el efecto de las vibraciones provocadas por el movimiento rotatorio del mismo. Entender el estado de la maquinaria a partir de su comportamiento vibratorio representa una ventaja con todos los métodos destructivos o que implican la detención del proceso de trabajo de la máquina.

Un prototipo de medición económico y de codigo abierto se desarrolló con el fin de responder a esta ultilidad mencionada

El equipamiento montado consta de un RaspberryPi 3 con el cual se procesaron los datos obtenidos por un sensor mpu650.

El software para la adquisición y procesamiento de los resultados se realizó en el lenguaje Python y es absolutamente abierto. Se puede obtener en github

Un instrumento de medición casero

Si bien las imprecisiones de la medición no se puede desestimar, la riqueza del trabajo es la posibilidad de realizar un sistema de medición sencillo y poderoso desde una casa

El sistema se puso a prueba para medir el movimiento vibratorio de un electrodomestico antiguo de hogar. Luego se trato de amortizar dicho movimiento con la utlización de unos soportes de un material flexible

Los resultados respondieron a lo esperado en cuanto a la amortiguación del sistema

Resultados de la atenuación de vibraciones para un estudio forzado Sensor mpu 6050 montado sobre un protoboard conectado al Raspberry pi 3
